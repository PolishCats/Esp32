ESP32 LDR Monitor

Plataforma IoT para monitoreo, alertas y reportes de luminosidad en tiempo real

v1.2.0 Docker Ready 2026

📋 Sobre el Proyecto

Solución web completa para visualizar datos enviados por un ESP32 con sensor LDR, administrar dispositivos y operar un panel de control centralizado. Incluye autenticación segura, configuración remota, generación de reportes y almacenamiento histórico para análisis de comportamiento lumínico.

🛠️ Tecnologías Utilizadas

Node.js Express.js MySQL HTML5 CSS3 JavaScript ES2022 Chart.js JWT bcrypt Nodemailer PDFKit Docker Compose ESP32 LDR Sensor

✨ Características

  • Autenticación segura con JWT y bcrypt
  • Dashboard con gráficos en tiempo real
  • Gestión de dispositivos ESP32 y API Keys
  • Gráfico histórico de 24 horas
  • Sistema de alertas configurables
  • Indicador de estado del ESP32
  • Exportación en CSV y PDF
  • Envío de reportes por correo
  • Limpieza automática de datos
  • Panel de configuración de rangos
  • Despliegue rápido con Docker Compose
  • Diseño responsive (móvil/escritorio)

🔌 Hardware Compatible

  • ESP32 (cualquier variante con ADC)
  • Sensor LDR (resistencia dependiente de luz)
  • Resistor de 10kΩ (divisor de voltaje)
  • Alimentación: 3.3V / 5V USB

Pin ADC: GPIO34 (o cualquier pin ADC del ESP32)
Resolución: 12 bits (0–4095)

🔗 API Endpoints

  • POST /api/auth/login – Iniciar sesión
  • POST /api/auth/register – Registro
  • POST /api/data – Enviar lectura (ESP32)
  • GET /api/dashboard/latest – Última lectura
  • GET /api/dashboard/historical – Histórico
  • GET /api/reports/csv – Descargar CSV
  • GET /api/reports/pdf – Descargar PDF

🚀 Instalación Rápida

1. Clonar repositorio y ubicarte en la carpeta raíz

2. Verificar que Docker Desktop esté iniciado

3. Ejecutar docker compose -f docker/docker-compose.yml up -d

4. Confirmar estado con docker compose -f docker/docker-compose.yml ps

5. Abrir http://localhost:3000

Conoce el código fuente en el repositorio oficial del proyecto: github.com/PolishCats/Esp32
