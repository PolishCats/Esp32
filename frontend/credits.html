ESP32 LDR Monitor

Sistema completo de monitoreo de sensor de luz basado en ESP32

v1.0.0 Open Source 2024

📋 Sobre el Proyecto

Plataforma web para visualizar en tiempo real los datos de un sensor LDR (Light Dependent Resistor) conectado a un microcontrolador ESP32. Permite monitorear la intensidad de luz ambiente, configurar alertas, generar reportes y mantener un historial de lecturas.

🛠️ Tecnologías Utilizadas

Node.js Express.js MySQL HTML5 CSS3 JavaScript ES2022 Chart.js JWT bcrypt Nodemailer PDFKit ESP32 LDR Sensor

✨ Características

  • Autenticación segura con JWT y bcrypt
  • Dashboard con gráficos en tiempo real
  • Gráfico histórico de 24 horas
  • Sistema de alertas configurables
  • Indicador de estado del ESP32
  • Exportación en CSV y PDF
  • Envío de reportes por correo
  • Limpieza automática de datos
  • Panel de configuración de rangos
  • Diseño responsive (móvil/escritorio)

🔌 Hardware Compatible

  • ESP32 (cualquier variante con ADC)
  • Sensor LDR (resistencia dependiente de luz)
  • Resistor de 10kΩ (divisor de voltaje)
  • Alimentación: 3.3V / 5V USB

Pin ADC: GPIO34 (o cualquier pin ADC del ESP32)
Resolución: 12 bits (0–4095)

🔗 API Endpoints

  • POST /api/auth/login – Iniciar sesión
  • POST /api/auth/register – Registro
  • POST /api/data – Enviar lectura (ESP32)
  • POST /api/data/simulate – Simular datos
  • GET /api/dashboard/latest – Última lectura
  • GET /api/dashboard/historical – Histórico
  • GET /api/reports/csv – Descargar CSV
  • GET /api/reports/pdf – Descargar PDF

🚀 Instalación Rápida

1. Clonar repositorio

2. Importar sql/schema.sql en MySQL

3. Copiar .env.example a .env y configurar

4. cd backend && npm install

5. npm start

6. Abrir http://localhost:3000
